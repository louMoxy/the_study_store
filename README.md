# the_study_store

To start gitea:
(in cmd - not gitbash)
cd gitea
gitea-1.2-windows-4.0-amd64.exe web

To start backbone:
cd client
npm install
npm start

gitea template files which have been modifies are:
repo/bare --- when there is no files in the repo)
repo/home --- repo list
user/settings/password --- password change response
